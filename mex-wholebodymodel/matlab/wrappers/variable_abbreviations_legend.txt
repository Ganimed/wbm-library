Legend of abbreviations and acronyms which are used for the variable names:

    B          ...  transformation matrix (rigid body)
    E          ...  transformation matrix (velocity)
    H          ...  homogeneous transformation (matrix)
    Mx         ...  Cartesian mass matrix (spatial mass matrix)
    R          ...  rotation (matrix)
    TF, T      ...  transformation (vector/matrix)
    X          ...  adjoint transformation (matrix)
    ac         ...  contact acceleration(s)
    acc        ...  acceleration(s)
    adj        ...  adjacency (matrix)
    arT        ...  angle rate transformation (matrix)
    arr        ...  array
    atf        ...  arms and torso free
    avT        ...  angle velocity transformation (matrix)
    b          ...  base
    bf         ...  base frame
    bgn        ...  begin
    bkgrd      ...  background
    c          ...  Cartesian OR contact OR constraint (**)
    c...       ...  cell array (begin of variable name)
    cX         ...  co-adjoint transformation (matrix)
    cap        ...  capture
    ccstr      ...  contact constraint
    cent       ...  centroidal
    centf      ...  centrifugal OR centrifugal force(s) (**)
    cfrict     ...  Coulomb friction
    chkfun     ...  check functions
    chn        ...  chain
    chnt       ...  chain tree
    cl         ...  contact link(s)
    clink/clnk ...  contact link
    clp        ...  contact link pose
    cm         ...  center of mass
    cmo        ...  column major order
    cmp        ...  compare
    cntfgl     ...  centrifugal
    cols       ...  columns
    config     ...  configuration
    cor        ...  Coriolis
    cstate, cs ...  contact state
    cstr       ...  constraint
    ctc        ...  contact
    ctrl, ctr  ...  control
    cvec       ...  column vector
    d...       ...  derivative (dot) of a variable (begin of variable name), like dv of v (= v_dot), etc.
    dflt, df   ...  default
    dim        ...  dimension
    dp         ...  data point
    dpts       ...  data points
    ds         ...  dimensions
    dstv       ...  derivative of state vector
    elb, el    ...  elbow
    elms       ...  elements
    ept, ep    ...  endpoint
    et         ...  elapsed time
    exc, ex    ...  exception
    fb         ...  floating base
    fcn        ...  function
    ffun       ...  fast functions
    fh...      ...  function handle OR file handle (begin of variable name) (**)
    fix        ...  fixed
    fk         ...  forward kinematic(s)
    fld        ...  field
    flnk       ...  fixed link
    fltb       ...  floating base
    fltg, flt  ...  floating
    fps        ...  frames per second
    frict      ...  friction
    frm        ...  frame
    ft         ...  foot
    fwd        ...  forward
    fwnd       ...  figure window
    g          ...  gravity acceleration (vector)
    gbl        ...  global
    gbl_...    ...  global variable (begin of variable name), e.g. "gbl_state", etc.
    gen        ...  generalized
    gp         ...  grid point(s) OR goal point (**)
    gr         ...  gripper
    grnd, grd  ...  ground
    grp        ...  group
    h...       ...  "handle" to an object (begin of variable name), e.g. hFig (handle to a figure object)
    ..._h      ...  "hat" (end of variable name), 'h' declares that this variable is an unit vector of length 1, e.g. "f1_h", "fcp_h", etc.
    h_c        ...  centroidal momentum (vector)
    hf         ...  half
    hgt        ...  height
    id         ...  identity
    idx        ...  index(es)
    ip         ...  index position
    jlim, jl   ...  joint limit(s)
    jnt        ...  joint
    jpos       ...  joint position(s)
    jspc       ...  joint space
    ker        ...  kernel (null space)
    lctn, loc  ...  location
    len        ...  length
    lgd        ...  legend
    lgt, lt    ...  light
    ll         ...  lower limit(s) OR left link (**)
    llnk       ...  left link
    lmts       ...  limits
    ln         ...  line
    lnk        ...  link
    lnkfr      ...  link frame (frame to a specified link in the current joint of the robot)
    lwr        ...  lower
    m...       ...  "member" (begin of variable name), 'm' declares that this variable is a private or protected member variable, e.g. "mwbm", "mstv", etc.
    ...m       ...  "matrix" (end of variable name), like "skewm", "adjointm", etc.
    mat        ...  matrix
    mbd        ...  multi body dynamics
    mg         ...  mesh grid
    mgC        ...  mesh grid coordinates matrix
    mom        ...  moment/momentum
    mov        ...  movie
    mult       ...  multiplier(s)
    ndof       ...  number degrees of freedom
    nflds      ...  number of fields
    noi        ...  number of iterations OR number of integrations (**)
    nw         ...  new world
    o          ...  orthogonal
    ofs        ...  offset(s)
    on         ...  orthonormal
    onb        ...  orthonormal basis
    op         ...  operator
    or         ...  orientation
    orig       ...  origin
    ow         ...  old world
    p          ...  position OR translation (vector) (**)
    p...       ...  pointer to an object/variable/array (begin of variable name, only in C++)
    param, prm ...  parameter
    params     ...  parameters
    pcstr...   ...  pointer to a constant null-terminated string (begin of variable name, C++)
    pld, pl    ...  payload
    plds       ...  payloads
    pload      ...  payload
    pos        ...  position(s)
    pris       ...  prismatic
    prms       ...  parameters
    prop       ...  propert(y/ies)
    pstr...    ...  pointer to a null-terminated string (begin of variable name, C++)
    pt         ...  point
    q_j/qj, q  ...  joint angle(s) (**)
    qt, q      ...  quaternion (**)
    quat       ...  quaternion
    r          ...  root (link) OR rotation (orientation) (**)
    rX         ...  (mixed) rate transformation (matrix)
    rax        ...  rotation axis
    rb         ...  rigid body
    rchd       ...  reached
    ref        ...  reference
    resv       ...  result vector (boolean)
    rev        ...  revolute
    ridx       ...  row index(es)
    rl         ...  right link
    rlnk       ...  root link OR reference link OR right link (**)
    rmo        ...  row major order
    rotm       ...  rotation matrix
    rpts       ...  repeats
    rtype      ...  representation type
    rvec       ...  row vector
    sc         ...  scalar OR scaled (**)
    scn        ...  scene
    secs       ...  sections
    sect       ...  section
    set        ...  setting(s) OR set (**)
    sf         ...  scale factor(s)
    shf, sh    ...  shift
    shp        ...  shape
    skewm      ...  skew-symmetric matrix
    spt, sp    ...  start point
    st         ...  state
    stm        ...  state matrix
    stp        ...  state parameter(s)
    str        ...  string
    stv        ...  state vector
    stvqT      ...  state vector-quaternion transformation [VQS-Transformation of the current state vector] (7x1-vector) (*)
    sz         ...  size(s)
    ..._t      ...  "transposed" (end of variable name), "_t" declares that a matrix or vector is transposed, e.g. "M_t", "v_t", etc.
    tfms       ...  transformations
    tform, tfm ...  transformation matrix
    tgts       ...  targets
    tidx       ...  time index OR task index (**)
    tl         ...  trajectory line
    tpos       ...  target position(s)
    tr         ...  translation(s) OR trace (**)
    trans      ...  translation(s)
    trgt, trg  ...  target
    trq        ...  torque
    trqs       ...  torques
    tt         ...  tool-tip
    ul         ...  upper limit(s)
    upr        ...  upper
    v          ...  vector
    vX         ...  (mixed) velocity transformation (matrix)
    vb         ...  volume body
    veT        ...  vector Euler-angle transformation [translation with Euler angles] (6x1-vector)
    vel        ...  velocit(y/ies)
    vfrict     ...  viscous friction
    vh...      ...  vector of handles to objects (begin of variable name), e.g. vhdp = vector handle of data point objects
    vid        ...  video
    vp         ...  viewpoint(s)
    vpos       ...  vector position [translation] (vector)
    vqT        ...  vector-quaternion transformation [VQS-Transformation] (7x1-vector) (*)
    vtcs       ...  vertices
    vtr        ...  vector translation (vector)
    vtx        ...  vertex
    vwpts      ...  viewpoints
    vxT        ...  vector axis-angle transformation [translation with axis angles] (7x1-vector)
    w          ...  world
    wbm        ...  whole body model
    wf, WF     ...  world frame
    wf_T_b     ...  transformation FROM the 'base' TO the 'world frame'
    wid        ...  width



 (*) ... Without scale factor S. This is the same as if scale S = 1.
(**) ... Depends on the context.
